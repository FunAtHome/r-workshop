Intro To customizing ‘ggplot’ graphs

  • color = (update point or line color)
  • fill =
  • line type
  • size
  • alhpa - transparentecy
ggplot(data = ci_np, aes(x= year, y= visitors)) +
  geom_line(color = "red", 
            linetype = "dashed")

ggplot(data = ci_np, aes(x= year, y= visitors)) +
  geom_point(color = "blue",
             pch =17,
             size = 4,
             alpha = 0.5)+
  geom_line(color = "red")
